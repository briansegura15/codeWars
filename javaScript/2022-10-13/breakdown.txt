Introduction
The first century spans from the year 1 up to and including the year 100, the second century - from the year 101 up to and including the year 200, etc.

Task
Given a year, return the century it is in.

Examples
1705 --> 18
1900 --> 19
1601 --> 17
2000 --> 20

P: what if we are given null or a string
R: should return the century that the year given is in
E:
given 2005 should return 20
given 2022 should return 21
given 1993 should return 19
P:
create a function that takes in an integer
